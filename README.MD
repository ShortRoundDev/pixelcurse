#Pixel curse#

##Dependencies##
requires ncurses

##Building##
run make

##Usage##
pixelcurse [iamge.bmp] -[bw|16|8]

##Notes##
*Only takes a8r8g8b8 32-bit bmp file. For best results, export image with Gimp
*-8 kind of sucks
*-16 REALLY sucks
*-bw gives best fidelity
